# Rainwater Harvesting System UI

## Project Structure

```
src/
├── components/
│   ├── ui/              # UI components
│   └── childcomponents/ # Child components
├── types/               # TypeScript types
└── styles/              # Global styles
hooks/                   # Custom hooks
lib/                     # Utilities
```

## Getting Started

```bash
npm install
npm run dev
```

## Environment

```
VITE_GOOGLE_MAPS_API_KEY=your_api_key_here
```
